## Giới thiệu chương trình
- Chương trình quản lý linh kiện viết bằng ngôn ngữ c++
- Chương trình có các chức năng như thêm sửa và xóa và lưu vào file để đảm bảo tính lâu dài của dữ liệu
## Chức năng
- Thêm linh kiện
- Sửa linh kiện
- Xóa linh kiện
- Xem thông tin linh kiện
- Tìm kiếm linh kiện
## Mô tả
- Mỗi class đại diện cho 1 loại linh kiện xác định
- Mỗi class có 1 file.h để khai báo
- Sửa thông tin thông qua mã số của linh kiện đó
- Dữ liệu được xử lý trong vector và lưu lại vào file trong mỗi lần mở máy vì dữ liệu ở vector sẽ mất sau khi tắt máy
- Sửa thông qua mã số linh kiện giúp xác định rõ chính xác linh kiện muốn sửa
- Mô hình của chương trình là xử lí dữ liệu ở vector sau đó lưu lại vào file
