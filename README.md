##Giới thiệu chương trình
- Chương trình quản lý linh kiện viết bằng ngôn ngữ c++
- Chương trình có các chức năng như thêm sửa và xóa và lưu vào file
# Chức năng
- Thêm linh kiện
- Sửa linh kiện
- Xóa linh kiện
- Xem thông tin linh kiện
- Tìm kiếm linh kiện
# Mô tả
- Mỗi class đại diện cho 1 loại linh kiện xác định
- Mỗi class có 1 file.h để khai báo
- Sửa thông tin thông qua mã số của linh kiện đó
